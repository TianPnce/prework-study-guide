HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1,h2, and h3: size of heading.
  • .html file extension indicates that this is a HTML file.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Refer to a color by its hexadecimal code, or hex #.
  • .css: file extension indicates that this is a CSS file.

Git

  • git status: checks what breanch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commit all files in the working branch.
  • git commit -m: associate a short descriptive message with our commit.
  • git push origin "BranchName": push all of our changes.
  • git pull origin main: make sure our current branch is up to date before opening a pull request.

JavaScript

  • .js: file extension indicates that this is a JavaScript file.
  • To decalre a variable: 'var' keyword.
  • All strings surrounded by quotation marks.
  • Each line of JavaScript code ends with a semicolon.
  • Console: refers to a test environment that developers use to check out code.
  • Chrome DevTools: keyboard commands Command+Option+I or right click on website hit "Inspect."
  • 'if' statement: will only execute if the statement meets a condition, or is true.
  • All values are truthy unless they are defined as falsy. That is, all values are truthy except false, 0, -0, 0n, "", null, undefined, and NaN.
  • '===' checks to see if two values are equal.
  • Array: a single variable that is used to hold a group of data.
  • First item in any array has an index of 0, not 1.
  • x++ is a shortcut way of writing x+1, x will increase by 1.
